AI Service Integration Test

This page tests the integration with Hugging Face AI services.

Important: To run this test successfully, you need to:
  1. Get a free Hugging Face API token from https://huggingface.co/settings/tokens
  2. Add it to your .env file as VITE_HF_API_TOKEN=your_token_here
  3. Restart the development server
